## 🎯 How to Use banana-claude

1. Open the app.

2. You will see a simple input box labeled “Enter your text prompt.”

3. Type a description of the image you want. For example, “A sunset over the mountains” or “A futuristic city at night.”

4. Press the “Generate” button.

5. The app will contact the Gemini API through Claude Code to create your image.

6. Wait a few seconds. Your generated image will appear below the input box.

7. Use the save button to download your image as a PNG or JPEG file.
